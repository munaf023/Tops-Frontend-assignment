Types of JavaScript Events:

Mouse Events:

Click: Occurs when the mouse button is clicked.
DoubleClick: Occurs when the element is double-clicked.
MouseDown, MouseUp: Occur when the mouse button is pressed down or released over an element.
MouseOver, MouseOut: Occur when the mouse pointer moves over or out of an element.
MouseMove: Occurs when the mouse pointer is moved while it is over an element.
MouseEnter, MouseLeave: Occur when the mouse pointer enters or leaves an element.

Keyboard Events:

KeyDown, KeyPress, KeyUp: Occur when a key on the keyboard is pressed down, pressed and released, or released, respectively.

Form Events:

Submit: Occurs when a form is submitted.
Reset: Occurs when a form is reset.
Focus, Blur: Occur when an element gains or loses focus.
